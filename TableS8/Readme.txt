-------------------------------------------------------------------------------------------------------
README file to replicate simulation results in Table S8
-------------------------------------------------------------------------------------------------------

* Step 1: Run the simulation codes in the folder "Sim_codes":
    ** BOIN12 can be run on a local laptop.
    ** Conventional Utility-Based (CUB), Strict Stagger, and the proposed Adaptive Stagger methods are recommended to be run on an HPC.

* Step 2: Once the results are obtained, run the code "result_agg_code_TableS8.R" to aggregate all results.
          The combined results (a CSV file) can be found in the folder "TableS8".
